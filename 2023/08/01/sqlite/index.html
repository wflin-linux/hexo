安装 sqlite3

  • sqlite3 这个是安装 sqlite3 数据库

  • libsqlite3-dev 这个安装 sqlite3 开发的套件（头文件和库等）

    sudo apt install sqlite3 libsqlite3-dev

验证

  • 查看版本

    sqlite3

常用指令

  • .help 查看帮助

  • .show 查看命令提示符的默认设置

  • .schema sqlite_master 查看表结构

  • .databases 查看数据

  • .tables 查看表

  • .header on

    .mode column

    .timer on

  • 增删改查和 mysql 语句的 sql 一样不再重复

c 语言连接 sqlite3

  • 代码 sqlite_test.c

  • 运行指令

    gcc sqlite_test.c -o sqlite_test -lsqlite3
